# IoT Environmental Monitoring System

This is the repository for my Bachelor's Diploma project (KPI, 2026).

The goal is to build a real-time data pipeline for environmental monitoring. The core stack for this project will include:

* **Python** (for data producers and consumers)
* **MQTT** (for data transport)
* **InfluxDB** (for time-series storage)
* **Grafana** (for data visualization)
* **Docker** & **Docker Compose** (for containerization and deployment)

*(This project is currently in the initial setup phase.)*
